Willkommen bei Banbury!

Für den Anfang haben wir hier Richtlinien geschaffen, die die Arbeit und vorallem Zusammenarbeit erleichtern sollen. Es wird darum gebeten diese Richtlinien zu beachten und sich so strikt wie möglich daran zu halten.

Inhalt

Dateien Programmierng Variablen Funktionen Debug Programmierung

1. Dateien

  • PHP, HTML und sonstige Text-Dateien sollen immer im UTF-8 Format vorliegen
  • Die globalen Konfigurationsdateien für das Projekt sind in /php/Config
    • Neue Konfigurationsdateien können hinzugefügt werden.
    • Sie werden in /php/Includes.php aufgerufen.
  • Die globalen Funktionen sind in /php/Functions.php
  • Jede Funktion, die das Projekt selbst bietet wird im Dateisystem von einem Unterordner abgebildet
    • Enthält den Ordner "Content" in dem HTML oder PHP Dateien für die Anzeige von Informationen bereitliegen
    • Enthält den Ordner "php" in dem Einzelne PHP Dateien mit speziellen Aufgaben liegen können
    • Enthält eine "index.php", deren Aufgabe die Vermittlung ist.

2. Programmierung

2.1 Variablen

  • Es gibt globale Variablen in PHP, nutze sie!
  • Es gibt globale Projektvariablen in Banbury, nutze sie!
  • Es gibt globale Konstanten in Banbury, nutze sie!
  • Es gibt Session-Variablen in Banbury, nutze sie!
  • Die Namensgebung von Variablen ist völlig freigestellt.

2.2 Funktionen

  • Funktionen können und sollen sogar soweit wie möglich hinzugefügt werden.
  • Die Implementierung dieser Funktionen soll so allgemein wie möglich gehalten werden.
  • Die Namensgebung von Funktionen erfolgt wie in folgenden Beispielen
    • Funktionsnamen beschreiben ihre Aufgabe
    • Sie sind Worte ohne Leerzeichen
    • Jedes Wort beginnt mit einem Großbuchstaben
    • Ausnahme sind Attribute wie: (unvollständige Liste)
      • r - rekursiv
      • a - assoziativ
      • i - caseinsensitive
    • Attribute werden zu einem Buchstaben gekürzt und an den Anfang des Funktionsnamen gestellt Bsp: aArrayIntoString

3. Debug

Zum Debuggen neuer oder alter Funktionen befindet sich unten auf der Seite ein Button mit der Aufschrift "Debug Info". Wenn man diesen drückt bekommt man Informationen über die Inhalte aller Wichtigen Arrays sowie über alle Queries während des Aufrufs. Es wird darum gebeten diesen Button zu nutzen. Der Inhalt der Debug Info Anzeige wird in /index.php generiert.
